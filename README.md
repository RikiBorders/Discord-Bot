# BornAgain
To activate the virtual env, run `source/BornAgain/.venv/bin/activate` (e.g source /home/rkabord/workspace/discordBot/BornAgain/.venv/bin/activate
)

To run the bot, run `sudo ./scripts/deploy.sh`

This project can also be deployed and managed using [homelab](https://github.com/RikiBorders/homelab)
